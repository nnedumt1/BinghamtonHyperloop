# BinghamtonHyperloop

#### Binghamton Hyperloop GNC code necessary to communicate with the Binghamton Hyperloop Pod
##### Server/Client code
  - Node.js
  - Express
##### Socket code
  - C



##### Web service run
  nodejs index.js \<PORT\>  
  nodejs index.js 3001

##### Web server run
  nodejs index.js \<PORT\> \<HOST:PORT\>  
  nodejs index.js 3002 http://localhost:3001
